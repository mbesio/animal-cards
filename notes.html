My HTML notes

Markup

You have to use markup to tell the browser what to do.
Otherwise all of what you write in the browser will be displayed as plain text.
In particular, you can use the em tag to write something in italics, for example:
Hello, this is an example of the em tag .
You have opening and close tags, that generally come in pairs, but not always.
Tags will come in angle brackets, these always come in pairs.

Tags

Now some interesting tags:

  • You will use the br tag to break a line.
  • You will use the p tag to have a paragraph element

Nesting

Another important thing for your text is to use appropriately nesting rules.
To use these rules in an appropriate way it is important to not overlap elements.
Nesting is generally fine, overlapping, not.

Inline vs. Block

There is the Inline vs. Block element.
div is the generic block element.
The div element is used a lot in HTML! So better be comfortable wtih it! span is the generic inline element.

Lists

For ul (unordered list) and ol (ordered list), you always have to have closing tags.
Instead, for li (elements of the list), you do not need to have the closing element. This is optional and can or cannot be there.

URL

URL stands for Uniform Resource Locator. When you write url in HTML, write the full form, the absolute URL, strating from http... Then :// Then a domain name Then a file path (can be empty or a single slash) Doria

Images
